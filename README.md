# Проект "Мой банк"

## Описание
Проект "Мой банк" - это небольшое приложение для банковских операций клиента

## Установка

1. Клонируйте репозиторий
git clone https://github.com/ksen6423/project7.git
2. Установите зависимости
pip install -r requirements.txt
 
## Использование

1. Зайдите в приложение
2. Откройте последние операции по карте
3. Выберите валюту, по которой нужно получить последние транзакции
